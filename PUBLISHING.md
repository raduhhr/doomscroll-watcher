# Publishing Notes

This repo is mainly set up for `Load unpacked`, but if you ever publish it, zip the extension with [manifest.json](/S:/PROJECTS/doomscroll-watcher/manifest.json) at the archive root.

## Current scope

- Instagram: all routes
- Facebook: all routes
- LinkedIn: all routes
- YouTube: `Shorts` only

## Store copy baseline

- Name: `Doomscroll Punisher`
- Short description:
  `Blocks scrolling on Instagram, Facebook, LinkedIn, and YouTube Shorts and interrupts the attempt with a timed warning modal.`

## Privacy / behavior summary

- No backend
- No account
- No remote code
- No intentional user-data collection
- Facts and media are bundled locally in the extension

## Submission reminder

If you publish, make sure the listing clearly says this is an unofficial personal productivity extension and not an official product from Meta, LinkedIn, Microsoft, YouTube, Google, or Chrome.
